# Kingdom of Israel Platform

A premium Sabbath bible study and community platform built with Next.js, TypeScript, and hand-authored CSS.

## Stack

- Next.js App Router
- React
- TypeScript
- CSS design tokens
- next/font Google typography

## Development

```bash
npm install
npm run dev
npm run build
npm run typecheck
```

The current Phase 1 foundation includes the public shell, KOI brand system, homepage scaffold, and multilingual lessons route.
